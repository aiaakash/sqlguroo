# ✨ Features

- 🤖 **AI-Powered SQL Generation**:
  - Convert natural language questions into accurate SQL queries
  - Support for complex queries with JOINs, aggregations, and subqueries
  - Context-aware query generation based on your database schema

- 📊 **Interactive Data Visualization**:
  - Multiple chart types: Bar, Line, Pie, Table, and Metric cards
  - Auto-suggested chart types based on query results
  - Customizable dashboards with drag-and-drop widgets

- 🔌 **Multi-Database Support**:
  - MySQL, PostgreSQL, ClickHouse, BigQuery, Snowflake
  - Redshift, Oracle, MSSQL, and more
  - Secure credential management with encryption

- 🧠 **Smart Schema Understanding**:
  - Automatic schema extraction and caching
  - Semantic table/column matching using embeddings
  - Schema documentation with RAG-powered assistance

- 💬 **Natural Language Analytics**:
  - Ask questions in plain English (or any language)
  - Get descriptive answers with insights and summaries
  - Choice between tabular results or narrative explanations

- 🛡️ **Built-in Safety & Governance**:
  - Read-only mode option for safe exploration
  - Query validation and dangerous operation blocking
  - Row limits and query timeouts to protect databases

- 📈 **Dashboard & Reporting**:
  - Save and organize queries into dashboards
  - Share dashboards with team members
  - Export results to CSV, JSON, or images

- 💾 **Query Management**:
  - Query history with full context
  - Save and categorize frequently used queries
  - Fork and modify existing queries

- 🌎 **Multilingual Support**:
  - English, 中文 (简体), 中文 (繁體), Español, Français, Deutsch
  - 日本語, 한국어, Português, Русский, Italiano, and more

- 💳 **Flexible Subscription Plans**:
  - **Free**: 5 queries to get started
  - **Pro**: 200 queries per month for professionals
  - **Ultra**: 650 queries per month for power users

- 👥 **Team Collaboration**:
  - Multi-user support with secure authentication
  - Role-based access control
  - Shared database connections and queries

- ⚙️ **Enterprise-Ready**:
  - Docker deployment support
  - Self-hosted option for data privacy
  - Configurable via environment variables

## 🐘 AI-Powered Database Analytics with SQL Guroo

SQL Guroo transforms how you interact with databases. Instead of writing complex SQL queries, simply ask questions in natural language and get instant insights. Whether you're a data analyst, business user, or developer, SQL Guroo makes database analytics accessible to everyone.

Connect to your existing databases securely, explore schemas visually, and generate reports without writing a single line of SQL. With support for all major databases and AI models, SQL Guroo brings the power of AI to your data stack.
